Description

This class represents an object mutation. It is used by mutators and mutation rules.

Internally the class has two mixin_collection objects - removing and adding that represent the mixins that are supposed to be removed and added by the mutation.

Additionally another mixin collection might be present - the source. It is a pointer that may be null. If it's not, it represents the mixins of the object that's been currently mutated.
