Macro BOOST_MIXIN_VERSION

BOOST_MIXIN_VERSION

Synopsis

// In header: <boost/mixin/version.hpp>

BOOST_MIXIN_VERSION

Description

The library's version.

It is composed of major*10000 + minor*100 + subminor. For example, version 2.31.11 would be 23111.
